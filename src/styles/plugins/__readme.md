# プラグインに対して使用するscss

## 決まりごと
- プラグイン毎に scss ファイルを増やして使用する。
- 基本的なデフォルトのスタイルを打ち消す際に使用する。
#### 例 :  
  ├── plugins/  
  │   ├── _caontctform7.scss  
  │   ├── _instagram-feed.scss  
  │   └── ...

- 基本的にすでにcssが当たっている状態だと思うので、上書きしていく形で書いてしまえば大丈夫です。
- main.scss ではデフォルトはコメントアウトしています。必要に応じてアンコメントアウトしてください。
